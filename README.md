# REDUX-101
Simple Redux example to test in NodeJS console.

## Steps
1) import redux
2) create store
3) create reducer and connect to store
4) create the initial state of the reducer
5) create the actions

## File and folder structure
- redux
  - /store
    - index.js
  - /reducer
    - index.js
  - /actions
    - actionTypes.js
    - index.js
- index.js

## Implementation
1) clone the repo
2) npm init
3) npm i redux
4) node index.js
